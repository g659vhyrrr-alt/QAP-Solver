2. Método de mejora local

Se prueban todos los intercambios posibles entre pares de máquinas. Para cada intercambio, se genera una nueva asignación, se calcula su costo total y se compara contra el costo actual.

Si ΔC < 0, el intercambio mejora la solución. Si ΔC ≥ 0, el intercambio no reduce el costo.

3. Evaluación de intercambios
